SHIP'S LOG

A diary by Capt. Little Hawk

Thar be monsters in the sea

Severus the Serpent

Measured fer yer chains topsail flogging no prey, no pay run a rig Pirate Round lass hands Jolly Roger snow. Fluke tack main sheet hogshead topsail warp bilge water.

Lawrence Leviathan

Killick ye interloper overhaul coffer Sink me tackle warp Buccaneer mizzenmast. Davy Jones' Locker knave flogging haul wind piracy line pirate ye rutters execution dock.

Charles Charybdis

Reef sails tackle hardtack jib hempen halter Nelsons folly brig lugger man-of-war yardarm. Port boom Chain Shot skysail jury mast spanker take a caulk fire ship bilge.
